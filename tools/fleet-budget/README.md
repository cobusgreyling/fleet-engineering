# fleet-budget

Roll up `budget_daily_tokens` from agent manifests and compare against `fleet-budget.md` team caps.

## Usage

```bash
npx @cobusgreyling/fleet-budget .
npx @cobusgreyling/fleet-budget . --json
```

Pair with [loop-cost](https://github.com/cobusgreyling/loop-engineering/tree/main/tools/loop-cost) when setting caps.
